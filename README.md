# ngpoint

A carousel for HTML content

## `npm install ngpoint`

## Usage

Import ngpoint in your .module.js and add it to dependencies.

```javascript
import ngPoint from 'ngpoint';

angular.module('app', [ngPoint]);
```

Then, use it in your template as such:
s

```html
<slide-deck>
    <slide>
        <img src="path-to-image" />
    </slide>
    <slide>
        <img src="path-to-image" />
    </slide>
    <slide>
        <img src="path-to-image" />
    </slide>
</slide-deck>
```
